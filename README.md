# Multi-user-shopping-cart-system
A Multi-User Shopping Cart System in assembly language supports login/signup using arrays, with 3-attempt security. Users browse products, add/remove items in cart, view cart details, and calculate total using price×quantity. Finally, users place order, get a bill, and cart is cleared with confirmation message.

First, the **Login & Signup system** allows users to create accounts and log in using stored username and password arrays. Second, **Invalid Login Handling** restricts access after 3 failed attempts for security. Third, **Browse Products** displays available items and prices stored in arrays. Fourth, **Add Items to Cart** lets users select products and quantities, updating existing entries if needed. Fifth, **Remove Items from Cart** allows users to decrease quantity or delete items. Sixth, **View Cart** shows selected products with quantities and prices. Seventh, **Calculate Total Bill** computes the cost using price × quantity and shows the final amount. Finally, **Place Order** generates the bill, confirms purchase, clears the cart, and displays a success message. These features together create a complete, structured shopping system using basic array and logic operations in assembly language.
